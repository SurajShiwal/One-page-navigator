Till now you relied only on your themes to create one page websites. It was difficult and confusing. Customizations were hard and options were limited.

Introducing - One Page Navigator for Visual Composer

Top Features
  • Give Unique ID to the Rows
  • Link From Any Visual Composer Element
  • Link From WordPress Menu
  • Smooth Scroll
  • Navigator UI Element


Minimum Requirements

  • WordPress 4.0 and above
  • Visual Composer 4.0 and above
  • If you have got Visual Composer as a part of your theme, ensure that Core Visual Composer plugin is not heavily modified
